# Etsy Listing Field Rules

Compiled from the Etsy Seller Handbook ("Keywords 101", listing optimization guides) and Etsy Help Center seller docs.

## Title
- Max **140 characters**; official guidance: keep it **under ~15 words**, short and scannable.
- Clearly name the item; put the most important traits (subject, color, material, size) upfront — shoppers only see the first few words in search results.
- Word order has no direct ranking effect, but clear titles earn clicks and popularity feeds back into ranking.
- Only include holiday/occasion/recipient words if essential to what the item IS; otherwise put them in tags.
- Avoid: ALL CAPS spam, repeated words, subjective words ("perfect", "beautiful" → move to description), price/shipping/sale info, special characters like %&$, starting with shop name.
- Pattern that works for POD decor:
  `{Design subject} {Material/Feature} {Product type}, {Style} {Use-case} Decor`

## Tags
- Exactly **13 tags**, each max **20 characters** (including spaces).
- Multi-word phrases are far stronger than single words ("solar garden light" > "solar"). Google rule of thumb: if no one would type it into Google, it shouldn't be a tag.
- Tags must be in the shop's language — foreign-language tags don't help search (Etsy auto-translates).
- Mix: 3–4 high-traffic head terms, 5–6 long-tail phrases, 2–3 recipient/occasion tags.
- Don't repeat exact words used in attributes/category (those already count as tags).
- No plurals duplicated (Etsy matches singular/plural), no misspellings, no irrelevant trending terms.
- Tag idea sources: subject, style/aesthetic (cottagecore, vintage…), recipient (dog mom, frenchie lover), occasion (housewarming, christmas), product synonyms (lantern/lamp/light), solution ("patio decor").

## Description
- First **~160 characters** = Google meta description. Lead with keywords + value, not greetings.
- Structure with short paragraphs / emoji bullets for scannability.
- Include: design, dimensions, materials, technique, features, scenarios/gift angles, care, package contents, production-variance note.
- Etsy also indexes descriptions for search — naturally reuse top keywords in the first paragraph.

## Materials
- Up to 13 free-text entries; searchable. List the actual materials (Plastic, PVC) plus notable components (solar panel, LED).

## Category & Attributes
- Choose the most specific (deepest) category — listings are auto-included in all parent categories, and each level acts like a tag. Look up the exact official path in `etsy-categories.md` (grep, don't read whole).
- Attributes (vary by category): most categories have Color, Holiday, Occasion; others add category-specific ones (Room, Orientation, Material…). Fill all that truly apply — they power sidebar filters and act like extra tags. Details in `attributes.md`.

## Photos & Alt Text
- Up to **20 photos**; first photo = search thumbnail and a search factor: no text overlays or collages, min 635×635 px (2000 px+ recommended), clean background.
- Alt text max ~250 characters; describe subject, product, colors plainly.
- More details on photo SEO and follow-up shot ideas in `search-optimization.md`.

## Variations & Personalization
- Up to 2 variation types (e.g., Size, Design Sides), each up to 70 options; can have per-option price.
- Personalization toggle: a buyer text field (max 1024-char instructions) — great for custom names/photos on POD.

## Renewal / other fields the user sets manually
- Price, quantity, processing time, shipping profile, production partner disclosure (required for POD!), renewal options — remind the user that POD shops must disclose production partners under Etsy's Handmade Policy.
